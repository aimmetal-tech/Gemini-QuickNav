# Gemini QuickNav

一个专为 Google Gemini (网页版) 设计的浏览器插件，旨在帮助用户在长对话中快速定位和导航历史提问。

## 🛠️ 安装步骤

目前该插件处于开发阶段，请通过以下方式手动安装：

1. **下载代码**：
   ```bash
   git clone https://github.com/aimmetal-tech/Gemini-QuickNav.git
   ```
2. **打开扩展程序页面**：
   在 Chrome 或 Edge 浏览器地址栏输入 `chrome://extensions/` 并回车。
3. **开启开发者模式**：
   在页面右上角找到并开启 **“开发者模式”** 开关。
4. **加载插件**：
   点击左上角的 **“加载已解压的扩展程序”**，选择本项目所在的文件夹。

## 📖 使用说明

1. 访问 [Gemini 官网](https://gemini.google.com/)。
2. 进入任一对话页面，页面右下角出现一个蓝色的 **☰** 悬浮按钮。
3. 点击按钮展开 **“提问导航”** 面板。
4. 在面板中点击任何一条历史提问，即可快速跳转。
5. 点击面板顶部的 **↻** 按钮可手动刷新列表，点击 **×** 可收起面板。
